# 🛍️ Customer Shopping Trends Analysis – Data Analytics/ Data Science Portfolio Project

## 📌 Project Overview

This project is a complete **end-to-end Data Analytics Portfolio Project** that demonstrates industry-standard practices for analyzing **customer shopping trends** using real-world retail data.

The project covers the full data analytics pipeline — from data extraction and cleaning to advanced analysis and interactive dashboard visualization.

## 🚀 Technologies Used

* **SQL (PostgreSQL)** – Data querying, transformation, and aggregation
* **Python** – Data cleaning, preprocessing, and exploratory data analysis
* **Power BI** – Data visualization and dashboard creation

## 📊 Project Objectives

* Analyze customer purchasing behavior
* Identify key sales trends and patterns
* Discover insights related to:

  * Customer segments
  * Product categories
  * Regional sales performance
  * Profitability metrics
* Build an interactive dashboard for business decision-making
